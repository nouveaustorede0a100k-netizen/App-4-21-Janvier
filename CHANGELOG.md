# Changelog

Tous les changements notables de ce projet seront documentés dans ce fichier.

## [1.0.0] - 2024-01-XX

### Ajouté
- Vue quotidienne avec timeline des objectifs
- Gestion complète des catégories et sous-catégories
- Système de micro-objectifs avec complétion
- 4 types d'animations de progression (bar, circle, fill, grow)
- Notes quotidiennes contextuelles
- Vue calendrier mensuel
- 3 modes de progression (cumulative, weekly, monthly)
- Authentification Supabase complète
- Interface responsive (mobile + desktop)
- Flow de création de catégorie en 4 étapes
- Dark mode support
- Settings utilisateur avec préférences personnalisables

### Infrastructure
- Setup complet React + Vite + TypeScript
- Configuration Tailwind CSS
- Stores Zustand pour la gestion d'état
- Hooks personnalisés
- Services de calcul de progression
- Schéma Supabase avec RLS
